# 🪨 OGG Android Master Tool

**Oggchain - Mine, Stake, Build**

OGG Android Master Tool is the official cave-built mobile tool for the Ogg tribe.

It gives users a simple Android app to manage OGG directly from their phone — no Termux, no command line, no complicated setup.

Create a wallet, import a wallet, check your balance, send OGG, stake, claim rewards, manage cooldowns, and participate in the Tribe Pool from one cave dashboard.

---

## ⛏️ What You Can Do

### 🪨 Wallet

* Create a new OGG wallet
* Import an existing private key
* Save wallet locally with password protection
* View wallet address
* Copy wallet address
* Check OGG balance
* Send OGG
* Use slider or Max button when sending
* Export private key with password confirmation

### 🥩 Staking

* View staking dashboard
* Check total staked
* Check estimated APR
* View your staked OGG
* View pending rewards
* Stake OGG
* Unstake OGG
* Track cooldown amount
* See cooldown end time
* Withdraw after cooldown
* Claim staking rewards
* View your pool share

### 🏛️ Tribe Pool

* View Tribe Pool balance
* View proposal information
* Create proposals
* Vote YES or NO
* Finalize proposals
* Clean up vote history

### 🔍 Tools

* Open transaction explorer
* Open staking and tribe contract pages
* View contract addresses
* Open OGG Discord
* Open OGG Telegram
* View app logs

## 📲 Android Install

Download the latest APK from the **Releases** section.

Install:

```text
OGG-Android-v1.0.apk
```

Then open the app named:

```text
OGG
```

No extra folder is needed.
No Termux is needed.
No CLI setup is needed.

## 🔐 Important Safety Note

This is an early Android build.

Always save your private key safely.
If you lose your private key, your OGG cannot be recovered.
If someone else gets your private key, they can control your wallet.

Ogg say: keep cave key safe. 🪨
